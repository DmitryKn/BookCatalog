## Book Catalog Project

Full-stack project(MERN).
Used Tools:

- Front-end: JavaScript, HTML, CSS, React, Bootstrap, React-router
- Back-end: Node.js, Express, Mongoose(MongoDB),

# To start project:

1. In folder "backend"
   Type in console:

- npm i
- npm start

2. In main folder
   Type in console:

- npm i
- npm start
